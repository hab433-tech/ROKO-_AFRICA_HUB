# Security Policy

## Supported Versions
Security updates and patches are applied to active versions of the ROKO Africa Hub smart contracts and ecosystem apps.

| Version | Supported |
|----------|------------|
| v1.x     | ✅ |
| < v1.0   | ❌ |

---

## Reporting a Vulnerability
If you discover a security issue or potential exploit:

1. **Do not open a public issue.**
2. Contact the core team privately via email: **security@roko.africa**
3. Include:
   - A clear description of the vulnerability
   - Steps to reproduce (if possible)
   - A proof-of-concept (if relevant)

---

## Audit & Verification
ROKO Africa Hub will undergo external smart contract audits before major releases.  
All audits and bytecode verifications will be published in this repository for transparency.

---

## Responsible Disclosure
We value community researchers and ethical hackers. Responsible disclosures that lead to meaningful fixes may receive community recognition or rewards.
